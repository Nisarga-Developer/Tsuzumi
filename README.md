## Self-Hosting

### Host Locally or on VPS

```sh
$ git clone https://github.com/nisarga-developer/Tsuzumi
$ cd Tsuzumi
```

* Rename `.env.example` to `.env` and fill it up with your `TOKEN`

```sh
$ pip3 install -r requirements.txt
$ python3 main.py
```
Enjoy!!
